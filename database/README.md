# Database (MySQL)

Run in order:

```sql
SOURCE schema.sql;
SOURCE seed.sql;
```

This creates all required `DCS Arena` tables:

- `users`
- `students`
- `captains`
- `admins`
- `sports`
- `teams`
- `registrations`
- `venues`
- `matches`
- `announcements`
- `attendance`
- `recruitment`
- `leaderboard`
